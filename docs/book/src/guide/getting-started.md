# Getting Started

There are two ways to use brink:

1. **As a CLI tool** -- compile `.ink` files to bytecode and play stories interactively in the terminal. See [Installation](./installation.md) and [The CLI](../cli/index.md).

2. **As a Rust library** -- embed the runtime in a game or application to drive ink stories programmatically. See [Installation](./installation.md) and [Quick Start](./quick-start.md).
